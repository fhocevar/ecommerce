# E-commerce Enterprise Frontend

Frontend React + TypeScript para o projeto de e-commerce enterprise.

## Stack

- React
- Vite
- TypeScript
- Axios
- React Router
- CSS moderno sem framework externo

## Como executar

```bash
npm install
npm run dev
```

Acesse:

```text
http://localhost:5173
```

## Backend esperado

```text
http://localhost:8081
```

## Fluxo implementado

- Login JWT
- Catálogo de produtos
- Carrinho local
- Checkout com Idempotency-Key
- Aprovação de pagamento
- Dashboard simples do pedido criado
